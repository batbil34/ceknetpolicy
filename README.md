Gizlilik Politikası / Privacy Policy

Bu uygulama, kullanıcıların kayıtlarına fotoğraf ve ilgili veriler eklemesine olanak tanır. This app allows users to add content such as photos and related data to their records.

Tüm kullanıcı içerikleri Apple’ın CloudKit servisi kullanılarak güvenli bir şekilde kullanıcının kişisel iCloud hesabında saklanır. All user content is stored securely using Apple’s CloudKit and is associated with the user’s personal iCloud account.

Uygulama geliştiricisi olarak, kişisel verileri kendi sunucularımızda toplamayız veya saklamayız. We do not collect or store personal data on our own servers.

iCloud üzerinde saklanan kullanıcı içeriklerine doğrudan erişimimiz yoktur. We do not have direct access to user content stored in iCloud.

Veriler, Apple’ın güvenli altyapısı üzerinden iletilir ve saklanır. Data is transmitted and stored using Apple’s secure infrastructure.

Uygulamada analiz, reklam veya üçüncü taraf takip hizmetleri kullanılmamaktadır. We do not use analytics, advertising, or third-party tracking services.

Kullanıcı verileri üçüncü taraflarla paylaşılmaz. User data is not shared with third parties.

Bu politika gelecekte değişirse, güncellemeler bu sayfada yayınlanacaktır. If this policy changes in the future, updates will be published on this page.
